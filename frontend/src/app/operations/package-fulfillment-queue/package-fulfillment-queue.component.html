Guidance

What to do next

  • Awaiting requests need stock reserved

    Open an awaiting request to begin batch selection and quantity allocation. FEFO is applied by default.

  • Drafts resume where you left off

    Saved package work keeps line-level picks and warehouse choices. Resume a draft to continue allocation.

  • Preparing packages are being committed

    Review selections, add override notes if needed, then lock the package.

  • Ready packages hand off to Dispatch

    Committed packages appear in the Dispatch Queue for handoff.
