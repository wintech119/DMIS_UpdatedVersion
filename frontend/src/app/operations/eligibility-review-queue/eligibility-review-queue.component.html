How to work the queue

  • Open a request awaiting action

    Use the detail screen to inspect the evidence and item list.

  • Record an approve, reject, or ineligible decision

    Rejected or ineligible decisions require a reason before submission.

  • Decided requests leave this queue

    Approved requests move to fulfillment; rejected or ineligible requests exit the workflow.
