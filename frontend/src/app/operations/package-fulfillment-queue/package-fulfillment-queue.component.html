What to do next

Awaiting requests need stock reserved.

Open a request to begin batch selection and quantity allocation.

Draft packages can be resumed from the queue.

Open a draft row to continue reservation work without starting over.

Preparing packages are being committed.

Review selections, add override notes if needed, then lock the package.

Ready packages hand off to Dispatch.

Once committed, the package appears in the Dispatch Queue for handoff. Dispatched and received packages move out of this active-work queue.
