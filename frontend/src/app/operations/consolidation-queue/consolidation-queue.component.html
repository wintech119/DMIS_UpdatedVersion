Guidance

Consolidation phases

  • Awaiting legs

    Shadow transfers are planned but source warehouses have not yet dispatched.

  • In transit

    Legs are en route to the staging hub.

  • Partially received

    Some legs have landed. Open the package to request a partial release if the operation cannot wait.

  • Ready to dispatch

    All legs received. The package moves to the Dispatch Queue for onward handoff.
