# MACHINE-READABLE FORMAT RULES (do not change heading levels):
#   ## GROUP:<CODE> <Label>            — defines a top-level Group
#   ### FAMILY:<CODE> <Label>          — defines a Family within a Group
#   - ITEM: <description>              — representative item in that Family
#
# CODE STRUCTURE REMINDER:
#   [GRP 2][FAM 2][ITEM 4][VRNT 1-3][SEQ 1-2]   max 13 chars, all UPPERCASE

## CODE CONSTRUCTION REFERENCE

### Segment rules summary

```
Position  Length  Rule
──────────────────────────────────────────────────────────────────────
GRP       2       Exact code from GROUP table above (GR/HE/RE/SU/WS)
FAM       2       Exact code from FAMILY table under that GROUP
ITEM      4       Consonant-dominated 4-char abbreviation of primary noun
VRNT      1–3     Most important variant: capacity > material > size > generic GN
SEQ       1–2     Integer starting at 1; increment to avoid DB collision
──────────────────────────────────────────────────────────────────────
Total: 10–13 chars, all UPPERCASE, no separators
```
